# project
# Генератор случайных цитат (Random Quote Generator)

**Автор:** Афанасьеева Виталина

## Описание
Десктопное GUI-приложение для генерации случайных цитат из предопределённой или пользовательской коллекции.  
Приложение сохраняет историю сгенерированных цитат, позволяет фильтровать их по автору или теме, а также сохраняет все данные в JSON-файл.  
Реализовано на Python с использованием библиотек `tkinter` и `json`.

## Возможности
- Генерация случайной цитаты по нажатию одной кнопки.
- Список истории генераций (отображается время, автор, начало цитаты, тема).
- Фильтрация истории по **автору** или **теме** через выпадающие списки.
- Добавление новых цитат с проверкой на пустые поля.
- Автосохранение всех данных в файл `quotes.json`.
- Кроссплатформенность (Windows / macOS / Linux).

## Как запустить
1. Убедитесь, что установлен Python версии 3.6 или выше.
2. Сохраните файл `quote_generator.py` в любую папку.
3. Запустите приложение командой:
   ```bash
   python quote_generator.py
