# Генератор случайных цитат (Random Quote Generator)

**Автор:** Афанасьеева Виталина

## 📝 Описание
Приложение с графическим интерфейсом (Tkinter) для генерации вдохновляющих цитат. 
Позволяет:
- Случайным образом выбирать цитату из списка
- Сохранять все сгенерированные цитаты в историю
- Фильтровать историю по **автору** и по **теме**
- Добавлять **собственные цитаты** с проверкой на пустые строки
- Сохранять историю в JSON-файл (автозагрузка при запуске)
- Использовать приятную **розовую цветовую гамму**

## 🎨 Цветовая схема
- Фон окна: `#FFE4EC` (нежно-розовый)
- Кнопки: `#FF69B4` (ярко-розовый) и `#FFB6C1` (светло-розовый)
- Текст цитат: `#8B0045` (тёмный винно-розовый)
- Поля ввода: `#FFF0F5` (лавандово-розовый)
